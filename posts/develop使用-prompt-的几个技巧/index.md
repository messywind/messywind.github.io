# 使用 Prompt 的几个技巧


- 我是这个领域的小白，但我想要达到专家级的效果，这些是我的原始想法和参考素材，请你先不要生成内容，先消化这些信息，告诉我你理解了没有。

- 基于你刚刚学到的这些素材，如果我要完成这个任务，你需要我补充哪些背景、信息，或者利用你自身的知识，帮我把这些大白话翻译成结构清晰逻辑严密的 system prompt

- 尝试举例子而不是说理论。

- 尝试拆解问题到 prompt 链上，先说一点思路生成 prompt A，再根据 prompt A 生成 prompt B，最后根据 prompt B 生成最终结果。

- 告诉 AI 说你可以不知道，但请不要编造，如果我的数据没给全我可以补充，这样可以减少幻觉。

- 可以跟 AI 说一步一步拆解你的思路，增加思考过程。

- 注意合理运用多模态输入，例如图片、表格、视频等，以提高生成内容的质量和准确性。

- 大模型很熟悉 JSON。JSON 在网页、代码、API 文档中高频出现，属于它最熟悉的语法之一。输入 JSON 时，它不必揣测意图，直接把键值映射到输出即可，准确率随之上升。比如&#34;将图像转换为 JSON 提示符，包括大小和详细信息&#34;，可以将图片的 prompt 直接逆向出来自己生成。
